# 어드민 사이트 정책서 초안

## 📋 문서 개요

### 문서 정보
- **문서명**: 어드민 사이트 정책서 초안
- **작성일**: 2025년 11월 19일
- **버전**: 1.0
- **작성 목적**: 메시징 플랫폼 어드민 사이트 구축을 위한 기능 및 정책 정의

### 문서 범위
본 문서는 메시징 플랫폼의 어드민 사이트에 포함될 모든 기능과 정책을 정의합니다.
일반 사용자 사이트의 모든 기능에 대한 관리 및 모니터링 기능을 포함합니다.

---

## 1. 어드민 사이트 개요

### 1.1 목적
- 시스템 전반의 운영 및 관리
- 사용자 관리 및 지원
- 발송 내역 모니터링 및 통계 분석
- 정책 설정 및 시스템 설정 관리
- 법적 규정 준수 모니터링

### 1.2 주요 역할
- **시스템 관리자**: 전체 시스템 설정 및 관리
- **운영 관리자**: 일상적인 운영 업무 처리
- **고객 지원**: 사용자 문의 및 지원
- **심사 관리자**: 발신번호 승인, 템플릿 검수 등

### 1.3 접근 권한
- **최고 관리자**: 모든 기능 접근 가능
- **운영 관리자**: 운영 관련 기능 접근
- **심사 관리자**: 승인/검수 관련 기능만 접근
- **고객 지원**: 사용자 조회 및 지원 기능만 접근

## 2. 사용자 관리

### 2.1 회원 관리

#### 2.1.1 회원 목록 조회
**기능 설명:**
- 전체 회원 목록 조회 및 관리
- 회원 유형별 필터링 (개인/기업)
- 가입일, 상태별 정렬 및 검색
- 회원 상세 정보 조회

**주요 항목:**
- 회원 ID (이메일)
- 회원명 (개인명/기업명)
- 회원 유형 (개인/기업)
- 가입일
- 최종 로그인 일시
- 계정 상태 (정상/정지/탈퇴)
- 잔액 정보
- 발신번호 등록 개수
- 총 발송 건수

**검색/필터 기능:**
- 회원명/이메일 검색
- 회원 유형 필터
- 가입일 기간 필터
- 계정 상태 필터
- 발송 건수 범위 필터

**액션:**
- 회원 상세 정보 조회
- 계정 정지/해제
- 계정 삭제 (탈퇴 처리)
- 잔액 조정
- 발송 내역 조회
- 발신번호 관리 내역 조회

#### 2.1.2 회원 상세 정보
**기능 설명:**
- 개별 회원의 상세 정보 조회 및 관리

**표시 정보:**
- **기본 정보**
  - 회원 ID, 회원명, 회원 유형
  - 연락처 (전화번호, 이메일)
  - 가입일, 최종 로그인 일시
  - 계정 상태

- **기업 정보** (기업 회원의 경우)
  - 사업자등록번호
  - 대표자명
  - 사업자등록증 파일
  - 업종, 주소

- **계정 정보**
  - 잔액
  - 총 충전 금액
  - 총 사용 금액
  - 보너스 포인트

- **활동 정보**
  - 총 발송 건수
  - 등록된 발신번호 개수
  - 등록된 템플릿 개수
  - 주소록 그룹 개수

**관리 기능:**
- 계정 상태 변경 (정상/정지/탈퇴)
- 비밀번호 초기화
- 잔액 조정 (충전/차감)
- 보너스 포인트 지급/차감
- 메모 작성 (내부 관리용)

#### 2.1.3 계정 관리
**기능 설명:**
- 회원 계정 상태 관리 및 제재

**계정 정지:**
- 정지 사유 선택
- 정지 기간 설정 (무기한/기간 지정)
- 정지 시 발송 기능 제한
- 정지 해제 기능

**계정 삭제:**
- 탈퇴 처리
- 데이터 보관 정책 적용
- 개인정보 삭제 일정 관리

**잔액 조정:**
- 수동 충전/차감
- 조정 사유 입력
- 이력 기록

### 2.2 권한 관리

#### 2.2.1 관리자 계정 관리
**기능 설명:**
- 어드민 사이트 접근 관리자 계정 관리

**관리 항목:**
- 관리자 ID, 이름
- 권한 그룹 (최고 관리자/운영 관리자/심사 관리자/고객 지원)
- 생성일, 최종 로그인 일시
- 계정 상태

**권한 그룹별 기능:**
- **최고 관리자**: 모든 기능 접근
- **운영 관리자**: 사용자 관리, 발송 모니터링, 통계 조회, 정책 설정
- **심사 관리자**: 발신번호 승인, 템플릿 검수, 서류 검토
- **고객 지원**: 사용자 조회, 문의 처리, 발송 내역 조회

**액션:**
- 관리자 계정 생성/수정/삭제
- 권한 그룹 변경
- 비밀번호 초기화
- 접근 로그 조회

#### 2.2.2 접근 로그 관리
**기능 설명:**
- 관리자 접근 로그 기록 및 조회

**기록 항목:**
- 관리자 ID
- 접근 일시
- 접근 IP
- 접근 기능/페이지
- 수행 작업
- 결과 (성공/실패)

**조회 기능:**
- 기간별 조회
- 관리자별 필터
- 기능별 필터
- IP별 필터
- 검색 기능

## 3. 발신번호 관리

### 3.1 발신번호 승인 관리

#### 3.1.1 승인 대기 목록
**기능 설명:**
- 사용자가 등록 신청한 발신번호의 승인/반려 처리

**표시 정보:**
- 신청일시
- 회원 정보 (회원명, 회원 유형)
- 발신번호
- 통신사
- 명의자
- 명의 구분 (본인/타인/타사)
- 회원 유형 (개인/기업)
- 상태 (승인대기)

**검색/필터:**
- 회원명/발신번호 검색
- 신청일 기간 필터
- 회원 유형 필터
- 명의 구분 필터

#### 3.1.2 발신번호 상세 검토
**기능 설명:**
- 발신번호 등록 신청 상세 정보 조회 및 서류 검토

**표시 정보:**
- **신청 정보**
  - 회원 정보 (상세)
  - 발신번호
  - 발신번호 용도
  - 통합용 사용 여부
  - 명의 구분
  - 연락처 정보

- **서류 정보**
  - 본인인증 정보 (휴대폰 번호인 경우)
  - 제출 서류 목록
    - 통신서비스 이용증명원/통신사 가입증명서
    - 사업자등록증 (기업)
    - 재직증명서
    - 위임장 (타인/타사 명의)
    - 신분증 (타인 명의)
  - 서류 파일 다운로드 및 미리보기

**검토 기능:**
- 서류 확인 및 검증
- 승인 처리
- 반려 처리 (반려 사유 입력)
- 보완 요청 (추가 서류 요청)
- 메모 작성 (내부 검토 메모)

**승인 처리:**
- 승인 완료 시 상태 변경
- 승인 완료 알림 발송 (SMS/이메일)
- 발송 서비스 사용 가능 처리

**반려 처리:**
- 반려 사유 선택/입력
- 반려 알림 발송 (SMS/이메일)
- 재신청 가능 처리

#### 3.1.3 발신번호 전체 목록
**기능 설명:**
- 승인 완료된 모든 발신번호 목록 조회 및 관리

**표시 정보:**
- 발신번호
- 회원 정보
- 통신사
- 명의자
- 등록일
- 승인일
- 상태 (승인완료/정지/삭제)
- 사용 여부 (사용중/미사용)

**관리 기능:**
- 발신번호 정지/해제
- 발신번호 삭제
- 사용 이력 조회
- 발송 내역 조회

### 3.2 발신번호 정책 관리

#### 3.2.1 정책 설정
**기능 설명:**
- 발신번호 등록 정책 설정

**설정 항목:**
- 등록 개수 제한 (개인/기업별)
- 승인 처리 기간 설정
- 서류 유효 기간 설정
- 재신청 제한 기간
- 정지/삭제 정책

## 3.3 발신프로필 관리

### 3.3.1 발신프로필 목록 조회
**기능 설명:**
- 사용자가 등록한 카카오톡 채널(발신프로필) 목록 조회 및 관리

**표시 정보:**
- 회원 정보
- 발신프로필 ID (@아이디 형태)
- 카카오톡 채널명
- 상태 (등록/검수중/활성/중단/차단)
- 브랜드메시지 사용여부 (Y/N)
- 담당자 휴대폰 번호
- 카테고리
- 등록일
- 관련 템플릿 개수

**검색/필터:**
- 회원명/발신프로필 ID 검색
- 상태별 필터 (전체/등록/검수중/활성/중단/차단)
- 등록일 기간 필터

**관리 기능:**
- 발신프로필 상세 정보 조회
- 발신프로필 상태 동기화 (카카오톡 채널과 동기화)
- 발신프로필 정지/해제
- 발신프로필 삭제 (템플릿이 없는 경우만)
- 관련 템플릿 목록 조회

### 3.3.2 발신프로필 상세 정보
**기능 설명:**
- 개별 발신프로필의 상세 정보 조회 및 관리

**표시 정보:**
- **기본 정보**
  - 발신프로필 ID (@아이디 형태)
  - 카카오톡 채널명
  - 등록일
  - 상태 (등록/검수중/활성/중단/차단)
  - 브랜드메시지 사용여부

- **담당자 정보**
  - 담당자 휴대폰 번호
  - 인증 여부

- **카테고리 정보**
  - 선택된 카테고리 목록 (최대 3개)

- **등록 이력**
  - 등록일시
  - 등록자
  - 상태 변경 이력

- **관련 템플릿**
  - 등록된 템플릿 개수
  - 템플릿 목록 조회

**관리 기능:**
- 발신프로필 상태 동기화
- 발신프로필 정지/해제
- 발신프로필 삭제 (템플릿이 없는 경우만)
- 관련 템플릿 관리 페이지로 이동

### 3.3.3 발신프로필 정책 관리
**기능 설명:**
- 발신프로필 등록 및 관리 정책 설정

**설정 항목:**
- 등록 개수 제한 (회원별)
- 카테고리 선택 제한 (최대 3개)
- 발신프로필 상태 동기화 주기
- 삭제 정책 (템플릿이 있는 경우 삭제 불가)

## 4. 템플릿 관리

### 4.1 알림톡 템플릿 검수

#### 4.1.1 검수 대기 목록
**기능 설명:**
- 사용자가 등록 신청한 알림톡 템플릿의 검수 처리

**표시 정보:**
- 신청일시
- 회원 정보
- 템플릿명
- 템플릿 코드
- 상태 (검수대기/검수중/승인완료/반려)

**검수 프로세스:**
1. 사용자가 발신프로필 선택 후 템플릿 등록 신청
2. 관리자 검수 대기 목록 확인
3. 발신프로필 정보 확인
4. 카카오 검수 요청 (카카오 비즈니스 API)
5. 카카오 검수 결과 확인
6. 승인/반려 처리

#### 4.1.2 템플릿 상세 검토
**기능 설명:**
- 템플릿 상세 내용 조회 및 검수

**표시 정보:**
- 템플릿명, 템플릿 코드
- 메시지 내용
- 변수 정보
- 버튼 정보
- 이미지 정보
- 카카오 검수 상태
- 카카오 검수 결과 (승인/반려 사유)

**검수 기능:**
- 템플릿 내용 확인
- 카카오 검수 요청
- 카카오 검수 결과 확인
- 승인 처리
- 반려 처리 (반려 사유 입력)
- 메모 작성

### 4.2 브랜드톡 템플릿 관리

#### 4.2.1 템플릿 목록 조회
**기능 설명:**
- 브랜드톡 템플릿 목록 조회 및 관리

**관리 기능:**
- 템플릿 상세 조회
- 템플릿 정지/해제
- 템플릿 삭제

### 4.3 문자 템플릿 관리

#### 4.3.1 템플릿 목록 조회
**기능 설명:**
- 사용자가 등록한 문자 템플릿 목록 조회

**관리 기능:**
- 템플릿 내용 확인
- 부적절한 템플릿 정지/삭제
- 템플릿 사용 통계 조회

## 5. 발송 관리

### 5.1 발송 내역 모니터링

#### 5.1.1 전체 발송 내역 조회
**기능 설명:**
- 모든 사용자의 발송 내역 통합 조회 및 모니터링

**표시 정보:**
- 발송일시
- 회원 정보
- 발신번호
- 메시지 타입 (SMS/LMS/MMS/알림톡/브랜드톡)
- 발송 건수
- 성공 건수
- 실패 건수
- 상태 (완료/진행중/실패)

**검색/필터:**
- 회원명/발신번호 검색
- 발송일 기간 필터
- 메시지 타입 필터
- 상태 필터
- 발송 건수 범위 필터

**액션:**
- 발송 상세 내역 조회
- 발송 취소 (진행중인 경우)
- 재발송 처리
- 발송 로그 다운로드

#### 5.1.2 발송 상세 내역
