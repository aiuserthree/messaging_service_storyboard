## 4. 핵심 기능 명세

### 4.1 성공/실패 건수 상세 리스트

#### 성공 내역 상세 리스트
- **기능**: 발송 내역 테이블의 성공 건수 클릭 시 해당 발송의 성공 내역 상세 리스트 표시
- **표시 항목**: 수신번호, 발송일시, 상태, 결과 메시지
- **페이징**: 1페이지당 10개 항목 표시
- **UI**: 테이블 행 아래 확장 형태로 표시, 페이징 UI 포함

#### 실패 내역 상세 리스트
- **기능**: 발송 내역 테이블의 실패 건수 클릭 시 해당 발송의 실패 내역 상세 리스트 표시
- **표시 항목**: 수신번호, 발송일시, 상태, 실패 사유
- **선택 재발송**: 
  - 각 항목별 체크박스 제공
  - 전체 선택 체크박스로 일괄 선택/해제
  - "전체 선택" 버튼
  - "선택 재발송" 버튼으로 선택한 항목만 재발송
- **UI**: 테이블 행 아래 확장 형태로 표시, 재발송 액션 버튼 포함

### 4.2 페이징 처리
- **성공 내역**: 1페이지당 10개 항목 표시
- **페이지네이션 UI**: 이전/다음 버튼, 페이지 번호, 현재 표시 범위 및 전체 건수 표시
- **동작**: 페이지 변경 시 해당 페이지 데이터만 렌더링
