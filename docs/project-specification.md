# 메시징 서비스 프로젝트 명세서

## 1. 프로젝트 개요

### 1.1 프로젝트 정의
- **프로젝트명**: 메시징 서비스 플랫폼 (Messaging Service Platform)
- **프로젝트 목적**: 개인 및 기업 회원이 문자(SMS/LMS/MMS)와 카카오톡(알림톡/브랜드톡) 메시지를 효율적으로 발송하고 관리할 수 있는 통합 메시징 서비스 플랫폼을 제공
- **핵심 가치**: 
  - 다양한 메시지 타입 지원 (SMS/LMS/MMS, 알림톡/브랜드톡)
  - 법적 규정 준수 (정보통신망법, 공직선거법 등)
  - 사용자 친화적인 인터페이스로 대량 발송 간편화
  - 실시간 발송 결과 추적 및 통계 제공
  - 안전한 결제 및 세금계산서 발행 지원

### 1.2 목표 사용자
- **주 사용자 그룹**: 
  - 개인 회원: 개인 사업자, 프리랜서, 소규모 사업자
  - 기업 회원: 중소기업, 대기업, 공공기관, 정당/후보자
- **사용자 페르소나**:
  - 마케팅 담당자: 대량 광고 메시지 발송, 고객 관리
  - 행정 담당자: 공지사항 발송, 이벤트 안내
  - 선거 캠페인 매니저: 공직선거 문자 발송
  - 고객 서비스 담당자: 알림톡을 통한 고객 안내
- **사용 시나리오**:
  - 이벤트 홍보를 위한 광고 문자 대량 발송
  - 주문/배송 알림을 위한 알림톡 자동 발송
  - 선거 캠페인을 위한 공직선거 문자 발송
  - 고객 관리 및 마케팅을 위한 주소록 관리

### 2.2 핵심 기능 명세

#### 기능 ID: F001
**기능명**: 일반문자 발송
**우선순위**: 필수
**기능 설명**: 
개인/기업 회원이 일반 문자메시지(SMS/LMS/MMS)를 발송할 수 있는 기능. 발신번호 선택, 수신번호 입력, 메시지 작성, 즉시/예약 발송을 지원

**사용자 스토리**:
- As a 마케팅 담당자
- I want 일반 문자를 발송하고 싶다
- So that 고객에게 안내 메시지를 전달할 수 있다

**기능 상세**:
- 입력: 발신번호, 수신번호(직접입력/주소록/엑셀), 메시지 내용, 발송 시간
- 처리: 수신거부 번호 자동 제외, 중복 번호 제거, 바이트 계산, 메시지 타입 자동 전환
- 출력: 발송 완료 안내, 발송 내역 페이지 이동

**수용 기준**:
- [ ] 발신번호 선택 시 승인 완료된 번호만 표시
- [ ] 수신번호 입력 시 수신거부 번호 자동 제외
- [ ] 메시지 내용 90바이트 초과 시 SMS → LMS 자동 전환
- [ ] 발송 전 잔액 확인 및 부족 시 충전 안내
- [ ] 발송 완료 후 발송 내역에 자동 등록

**UI/UX 요구사항**:
- 화면 구성: 좌측 입력 영역, 우측 미리보기 영역
- 인터랙션: 실시간 바이트 카운터, 드래그 앤 드롭 파일 업로드
- 피드백: 발송 진행률 표시, 성공/실패 토스트 메시지

**비즈니스 규칙**:
- 규칙 1: 발신번호는 승인 완료 상태만 사용 가능
- 규칙 2: 수신거부 번호는 자동으로 발송 대상에서 제외
- 규칙 3: 잔액 부족 시 발송 불가

**연관 기능**:
- 선행 기능: 발신번호 등록 및 승인
- 후행 기능: 발송 결과 조회
- 관련 기능: 템플릿 관리, 주소록 관리, 충전하기

#### 기능 ID: F002
**기능명**: 광고문자 발송
**우선순위**: 필수
**기능 설명**: 
정보통신망법에 따른 광고성 문자메시지 발송 기능. (광고) 문구 자동 삽입, 080 수신거부 번호 필수, 발송 시간 제한 준수

**사용자 스토리**:
- As a 마케팅 담당자
- I want 광고 문자를 발송하고 싶다
- So that 상업적 홍보 메시지를 고객에게 전달할 수 있다

**기능 상세**:
- 입력: 발신번호, 수신번호, (광고) 옆 문구, 080 수신거부 번호, 메시지 내용, 발송 시간
- 처리: (광고) 문구 및 입력 문구 삽입, 080 수신거부 번호 삽입, 발송 시간 제한 검증
- 출력: 발송 완료 안내, 법적 확인 내역 저장

**수용 기준**:
- [ ] 발신번호 선택 시 번호만 표시 (용도 제외)
- [ ] (광고) 옆 문구 입력 필드 제공 (필수)
- [ ] 080 수신거부 번호 입력 필드 제공 (필수)
- [ ] 입력한 문구가 (광고) 다음 줄에 자동 삽입
- [ ] 080 수신거부 번호 입력 시 미리보기에 자동 반영
- [ ] 평일 08:00~21:00 외 시간 발송 차단
- [ ] 휴일 발송 차단
- [ ] 법적 확인 체크박스 필수 선택

**UI/UX 요구사항**:
- 화면 구성: 법적 안내사항 상단 표시, 발송 시간 제한 안내
- 인터랙션: 발송 시간 선택 시 제한 시간대 경고
- 피드백: 법적 규정 위반 시 명확한 오류 메시지

**비즈니스 규칙**:
- 규칙 1: 광고문자는 평일 08:00~21:00에만 발송 가능
- 규칙 2: 080 수신거부 번호 필수 입력 및 포함
- 규칙 3: (광고) 문구 및 옆 문구 필수 포함
- 규칙 4: 발신번호 표시 시 번호만 표시 (용도 제외)

**연관 기능**:
- 선행 기능: 발신번호 등록 및 080번호 연동
- 후행 기능: 발송 결과 조회
- 관련 기능: 수신거부 관리

#### 기능 ID: F003
**기능명**: 공직선거문자 발송
**우선순위**: 필수
**기능 설명**: 
공직선거법에 따른 선거운동 문자 발송 기능. 후보자/정당명 필수 포함, 금지어 검증, 080 수신거부 번호 입력

**사용자 스토리**:
- As a 선거 캠페인 매니저
- I want 공직선거 문자를 발송하고 싶다
- So that 유권자에게 선거운동 메시지를 전달할 수 있다

**기능 상세**:
- 입력: 발신번호, [선거] 옆 문구(후보자명/정당명), 선거운동정보(선택, 최대 20자), 080 수신거부 번호, 수신번호, 메시지 내용, 발송 시간
- 처리: [선거] 문구 및 후보자/정당명 삽입, 선거운동정보 입력 제한(20자/40바이트), 080 수신거부 번호 삽입, 금지어 검증, 선거운동 기간 확인, 발송 시간 제한, 수신번호 입력 필드 숫자만 허용
- 출력: 발송 완료 안내, 발송 기록 저장

**수용 기준**:
- [ ] 발신번호 선택 시 번호만 표시 (용도 제외)
- [ ] [선거] 옆 문구 입력 필드 제공 (필수, 플레이스홀더: "후보자명/정당명")
- [ ] 선거운동정보 입력 필드 제공 (선택, 최대 20자/40바이트, 실시간 바이트 카운터)

#### 화면 ID: S001
**화면명**: 일반문자 발송
**화면 목적**: 사용자가 일반 문자메시지를 작성하고 발송하는 화면

**화면 구성 요소**:
1. **헤더 영역**
   - 요소: 페이지 제목, 발신번호 등록 알림

2. **콘텐츠 영역**
   - 요소: 발신번호 선택, 수신번호 입력, 메시지 작성, 발송 시간 설정, 미리보기
   - 레이아웃: 좌측 입력 영역, 우측 미리보기 영역

3. **액션 영역**
   - 주요 액션: 발송하기, 테스트 발송
   - 보조 액션: 임시저장, 템플릿으로 저장

**인터랙션 명세**:
- 발신번호 선택 → 드롭다운에서 선택
- 수신번호 입력 → 직접입력/주소록/엑셀 선택
- 메시지 입력 → 실시간 바이트 카운터 업데이트
- 발송하기 클릭 → 발송 확인 팝업 호출

**화면 전환 규칙**:
- 진입 조건: 로그인 상태, 발신번호 승인 완료
- 이탈 조건: 발송 완료 후 발송 내역 페이지로 이동

   - 보조 액션: 그룹 추가, 일괄 작업
