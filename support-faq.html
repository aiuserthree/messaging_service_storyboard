결제/포인트 문자 발송 요금은 어떻게 되나요?

톡벨의 문자 발송 요금은 다음과 같습니다:

  • SMS (단문): 건당 8.4원 (90byte 이하)
  • LMS (장문): 건당 25원 (2,000byte 이하)
  • MMS (포토): 건당 55원 (이미지 첨부)

대량 발송 고객의 경우 별도 할인이 적용될 수 있습니다. 자세한 내용은 1:1 문의를 통해 상담 받으실 수 있습니다.

결제/포인트 포인트 충전은 어떻게 하나요?

포인트 충전은 다음과 같은 방법으로 가능합니다:

  • 신용카드: 실시간 충전
  • 계좌이체: 실시간 충전
  • 무통장입금: 입금 확인 후 충전 (평일 기준 1시간 이내)

마이페이지 > 포인트 충전 메뉴에서 원하시는 결제 방법을 선택하여 충전하실 수 있습니다.

문자발송 발신번호 등록 방법을 알려주세요.

발신번호 등록은 통신사 정책에 따라 사전 등록이 필요합니다:

  • 휴대폰 번호: 본인 인증 후 등록 (실시간)
  • 유선 전화번호: 통신사 가입 증명서 제출 후 등록 (1~2 영업일)
  • 대표번호 (1588, 1577 등): 대표번호 가입 증명서 제출 후 등록

마이페이지 > 발신번호 관리에서 등록하실 수 있습니다.

카카오톡 알림톡과 문자의 차이점이 무엇인가요?

알림톡과 문자는 다음과 같은 차이점이 있습니다:

  • 발송 채널: 알림톡은 카카오톡, 문자는 SMS/LMS/MMS
  • 발송 조건: 알림톡은 카카오톡 사용자에게만 발송 가능
  • 요금: 알림톡이 일반적으로 더 저렴 (건당 6원~)
  • 도달률: 알림톡은 카카오톡 앱 알림으로 더 높은 확인률

톡벨에서는 알림톡 발송 실패 시 자동으로 문자로 대체 발송하는 기능도 제공합니다.

회원/계정 회원가입은 어떻게 하나요?

톡벨 회원가입은 간단한 절차로 진행됩니다:

  • 홈페이지 우측 상단 '톡벨 가입하기' 클릭
  • 이메일 또는 휴대폰 번호로 본인 인증
  • 기본 정보 입력 (이름, 연락처, 비밀번호)
  • 이용약관 동의 후 가입 완료

가입 즉시 무료 테스트 포인트가 지급되어 바로 서비스를 이용해보실 수 있습니다.

문자발송 예약 발송은 어떻게 설정하나요?

예약 발송 설정 방법:

  • 문자 발송 화면에서 메시지 작성
  • 발송 옵션에서 '예약 발송' 선택
  • 원하는 발송 일시 설정
  • '예약 발송' 버튼 클릭

예약된 메시지는 발송 전까지 수정 또는 취소가 가능합니다. 발송내역 > 예약 발송에서 확인하실 수 있습니다.

카카오톡 카카오톡 발신 프로필은 어떻게 등록하나요?

카카오톡 발신 프로필 등록 절차:

  • 카카오톡 채널 개설 (카카오 비즈니스 센터)
  • 톡벨에서 발신 프로필 등록 신청
  • 카카오 검수 진행 (1~3 영업일)
  • 검수 완료 후 알림톡/브랜드 메시지 발송 가능

자세한 등록 가이드는 카카오톡 발송 > 발신 프로필 관리에서 확인하실 수 있습니다.

결제/포인트 세금계산서 발급이 가능한가요?

네, 세금계산서 발급이 가능합니다:

  • 포인트 충전 시 사업자 정보 입력
  • 충전 완료 후 자동 발급 (이메일 발송)
  • 마이페이지 > 결제내역에서 재발급 가능

개인 회원의 경우 현금영수증 발급이 가능합니다. 충전 시 휴대폰 번호 또는 현금영수증 카드번호를 입력해주세요.

기타 API 연동은 어떻게 하나요?

톡벨 API 연동 절차:

  • 마이페이지 > API 설정에서 API Key 발급
  • 개발자 문서 확인 (REST API 지원)
  • 테스트 환경에서 연동 테스트
  • 운영 환경 적용

PHP, Java, Python, Node.js 등 다양한 언어의 SDK를 제공하고 있습니다. 상세한 기술 문서는 개발자 센터에서 확인하실 수 있습니다.

문자발송 광고 문자 발송 시 주의사항은 무엇인가요?

광고 문자 발송 시 법적 필수 사항:

  • (광고) 문구 표기 필수 (메시지 시작 부분)
  • 수신거부 안내: "무료거부 080-XXX-XXXX" 표기
  • 발송 시간 제한: 오후 9시 ~ 오전 8시 발송 금지
  • 수신 동의자에게만 발송: 사전 동의 필수

톡벨 광고문자 발송 기능 사용 시 필수 문구가 자동으로 포함됩니다.

원하는 답변을 찾지 못하셨나요?

1:1 문의를 통해 더 자세한 상담을 받으실 수 있습니다.
