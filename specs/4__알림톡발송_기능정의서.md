## 주요 검증 사항

### 1. 발신프로필 검증
- 발신프로필 등록 및 승인 상태 확인
- 발신프로필 ID 형식 검증 (@아이디 형태)
- 중복 발신프로필 등록 방지
- 카테고리 정합성 확인 (최대 3개)
- 브랜드메시지 사용여부 확인
- 발신프로필 상태별 처리 (등록/검수중/활성/중단/차단)

### 2. 카카오톡 채널 및 템플릿 검증
- **템플릿 존재 여부 확인 (페이지 진입 시 최우선)**
- 템플릿 승인 상태 확인
- 템플릿 내용 수정 불가 확인
- 선택한 프로필과 템플릿 매칭 확인
- **템플릿 부재 시 발송 기능 비활성화**

### 3. 변수 검증
- 필수 변수 입력 확인
- 변수 형식 검증 (숫자, 날짜, 텍스트 등)
- 변수 값 길이 제한 확인
- 엑셀 업로드 시 변수 매핑 정확성 확인

### 4. 수신번호 검증
- 전화번호 형식 검증
- 중복 번호 제거
- 카카오톡 가입 여부 확인
- 수신 차단 사용자 확인

### 5. 대체 메시지 검증
- 대체 메시지 내용 필수 확인
- 변수 매핑 정확성 확인

### 6. 비용 검증
- 잔액 확인
- 충전 필요 시 안내

### 7. 사용자 경험
- **발신프로필 등록 유도 및 안내**
- **템플릿 부재 시 명확한 안내 및 등록 유도**
- 실시간 미리보기 (알림톡 형태)
- 변수 입력 가이드 제공
- 템플릿 기반 자동 생성 기능
- 테스트 발송 기능
- 명확한 오류 메시지
- 발송 상태 실시간 추적

## 참고 사항

### 발신프로필(카카오톡 채널) 관리
- 카카오톡 비즈니스 센터에서 채널 먼저 생성 필요
- 발신프로필 ID는 @ 포함 형태로 입력 (예: @조도램)
- 담당자 휴대폰 번호 입력 (선택사항, 인증요청 기능 포함)
- 카테고리 최대 3개까지 선택 가능
- 카테고리는 카카오톡 채널 생성 시 선택한 카테고리와 동일하게 설정
- 발신프로필 상태 관리
  - 등록: 최초 등록 상태
  - 검수중: 카카오 검수 진행 중
  - 활성: 사용 가능 상태
  - 중단: 일시 중단 상태
  - 차단: 사용 불가 상태
- 브랜드메시지 사용여부 설정 가능
- 발신프로필별 템플릿 매핑 관리
- 발신프로필 삭제는 관련 템플릿이 없는 경우에만 가능

### 카카오톡 알림톡 규정
- 카카오톡 채널(구 플러스친구) 사전 등록 필수
- 템플릿 사전 검수 및 승인 필수
- 템플릿 내용 수정 불가 (승인된 내용 그대로 사용)
- 변수 치환 기능 활용
- 정보성 메시지만 발송 가능
- 광고성 메시지는 브랜드톡 사용
- 발송 시간 제한 없음 (24시간 발송 가능)

### 템플릿 등록 프로세스
1. 카카오톡 채널 개설 및 등록
2. Tokbell에 발신프로필 등록
3. 템플릿 관리 페이지에서 템플릿 작성
4. 템플릿 검수 요청
5. 카카오 검수 진행 (1~2 영업일)
6. 승인 완료 후 알림톡 발송 가능

### 템플릿 부재 시 사용자 안내 사항
- **명확한 상태 표시**: "템플릿이 없어 발송할 수 없습니다"
- **등록 방법 안내**: 간단한 등록 절차 설명
- **원클릭 이동**: "템플릿 등록하러 가기" 버튼으로 즉시 이동
- **도움말 제공**: 템플릿 등록 가이드 링크
- **예상 소요 시간**: "템플릿 승인까지 1~2 영업일 소요" 안내

### 기술적 고려사항
- 대량 발송 시 성능 최적화
- 변수 치환 처리 로직
- 카카오톡 API 연동
- 발송 상태 실시간 추적 시스템
- 통계 및 로그 저장
- 엑셀 데이터 파싱 및 검증
- **템플릿 존재 여부 캐싱 처리 (성능 최적화)**
- **프로필 변경 시 템플릿 재확인 로직**
- **발신프로필 상태 동기화 (카카오톡 비즈니스 센터)**
- **발신프로필별 템플릿 매핑 관리**

### 대체 메시지 설정
- 알림톡 발송 실패 시 자동으로 SMS/LMS로 전환
- 변수 데이터 공유 및 매핑

### 연동 페이지
- **발신프로필 관리 페이지** (신규 추가)
- **카카오톡 템플릿 관리 페이지** (템플릿 부재 시 이동)
- 주소록 관리 페이지
- 발송내역 페이지
- 예약내역 페이지
- 충전하기 페이지

### 외부 연동
- **카카오톡 채널** (https://center-pf.kakao.com)
  - 비즈니스 채널로 등록된 카카오톡 채널(발신프로필)만 등록 가능
  - 발신프로필 ID 확인
  - 템플릿 등록 및 관리

### 알림톡 vs 광고문자 차이점
| 구분 | 알림톡 | 광고문자 |
|---|---|---|
| 플랫폼 | 카카오톡 | SMS/LMS/MMS |
| 사전 승인 | 템플릿 승인 필요 | 불필요 |
| 내용 수정 | 불가 (변수만 치환) | 자유롭게 작성 |
| 발송 시간 | 24시간 가능 | 평일 08:00~21:00 |
| 광고성 문구 | 불필요 | (광고) 필수 |
| 수신거부 | 카카오톡 내 차단 | 080 번호 필수 |
| 용도 | 정보성 메시지 | 광고성 메시지 |
| 비용 | 건당 N원 | SMS/LMS/MMS 별도 |
