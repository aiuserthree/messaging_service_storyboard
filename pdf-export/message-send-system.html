메시지 발송 시스템

문자 및 카카오톡 메시지 대량 발송 시스템 · 문서 버전 1.0

본 문서는 웹 애플리케이션 메시지 발송 시스템의 개요와 주요 기능을 PDF로 정리한 자료입니다. (참고: Next.js 기반 페이지의 초기 HTML 구조·메타 정보를 반영)

1. 목적

SMS·LMS·MMS 등 문자 메시지와 카카오톡 비즈메시지(알림톡·브랜드 메시지 등)를 대량·예약 발송할 수 있도록 지원하는 통합 발송 환경을 제공합니다.

2. 주요 기능 영역

  • 대량 발송 — 수신자 목록(엑셀 등) 기반 일괄 전송
  • 채널 연동 — 문자(SMS/LMS/MMS) 및 카카오톡 메시지 채널
  • 템플릿·변수 — 반복 발송을 위한 메시지 템플릿 및 치환
  • 발송 결과 — 전송 상태·실패 사유 조회
  • 인증·권한 — 로그인 기반 접근 제어(앱 구조상 AuthProvider·Layout 사용)

3. 시스템 구성(참고)

구분 설명
프론트엔드 Next.js(App Router), 클라이언트 렌더링 구간 포함
UI 전역 레이아웃(GNB), 알림 모달 컨테이너 등
문서 메타 제목: 메시지 발송 시스템 / 설명: 문자 및 카카오톡 메시지 대량 발송 시스템

4. 유의사항

실제 화면은 JavaScript 로딩 후 표시됩니다. PDF는 인쇄용 정적 요약본이며, 운영 정책·약관·요금은 서비스 최신 공지를 따릅니다.
